Negrito

- Para criarmos um texto em negrito utilizamos a tag strong

Itálico

- Para criarmos um texto em italico utilizamos a tag em

Sublinhado

- Para criarmos um texto sublinhado usamos as tags u ou ins

Tachado

- Para deixa um texto tachado usamos a tag del

Marcado

- Para deixarmos um texto em marcado usamos a tag mark

Menor

- Para exibir um texto em formato menor usamos a tag small

Subscrito

- Para exibir um texto em formato subscrito usamos a tag sub

Sobrescrito

- Para exibir um texto em formato sobrescrito usamos a tag sup

Quebra de linha

- Para quebrarmos uma
linha
, usamos a tag br

Abreviação

- Para criarmos uma abreviação, por exemplo, CPS, usamos a tag abbr e o atributo title

Tempo

- Para indicarmos um tempo, por exemplo, 10:00, usamos a tag time

Código

- Para indicarmos um texto como código usamos a tag code

Teclado

- Para indicarmos uma ação do usuário no teclado, por exemplo, CTRL + C, usamos a tag kbd

Citação

- Para inicarmos uma citação, usamos a tag q

Citação em bloco

- Para fazermos uma citação em bloco, por exemplo:

Lorem ipsum dolor, sit amet consectetur adipisicing elit. Veritatis, dolor quia! Cumque eveniet sit quisquam impedit, beatae aspernatur molestias harum nihil ipsam? Possimus at laboriosam dolor dolorem, amet provident veniam? Lorem ipsum, dolor sit amet consectetur adipisicing elit. Perferendis, officiis earum soluta repellendus incidunt, eos sint dicta blanditiis maxime exercitationem nobis, suscipit nostrum tempora vitae nesciunt ullam laudantium assumenda nisi?
usamos a tag blockquote

Endereço

- Para indicarmos um endereço, por exemplo:

Escrito por Vanessa Freitas
E-mail: vlfreitas.paws@gmail.com
Telefone: +5513999980877
Endereço: Rua 1, bairro 2
Brasil Usamos a tag address

Barra Escalar

- Para criarmos uma barra escalar, por exemplo:
Uso do disco local C: 2 de 10
Uso do disco local D: 60%

Usamos a tag meter e os atributos: value, min e max.

Barra de Progresso

- Para criarmos uma barra de progresso, por exemplo:

Baixando 32%

Usamos a tag progress e os atributos:value e max
